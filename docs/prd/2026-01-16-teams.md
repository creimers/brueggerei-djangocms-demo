# events

App to manage teams

## datamodel

### Team

- name
- slug (auto-generated from name)
- description
- team leads (many-to-many to User)
- image (django filer image field)

### user extension

- teams (many-to-many to Team)
- image (django filer image field)
- short bio

## django cms plugins

### Team Plugin

Displays a team with image, description and team leads (image, name, telephone or email)

### User Plugin

Displays a user with image and short bio.

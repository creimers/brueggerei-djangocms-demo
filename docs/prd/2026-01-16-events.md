# events

App to manage events. 

## datamodel

### Event
- title
- slug (auto-generated from title)
- organizing_team (foreign key to Team)
- organizing_user (foreign key to User)
- location (text field, default 'Br√ºggerei')
- short description
- full description
- image (django filer image field)
- auto advertise (default true)
- recurrences
- start time
- end time

use https://github.com/jazzband/django-recurrence for recurrences field

## django cms plugins

### Upcoming Events Plugin

Displays a list of upcoming events. Accepts 'number of events' field to limit number of displayed events.

## app hooks

### Events App Hook

Provides event listing and detail views at a specified URL.
